# Homebrew Tap

This repository hosts Homebrew formulae and casks for Ernest Gonzales apps.

## Usage

```bash
brew tap ernest-gonzales/homebrew-tap
brew install codex-tracker
brew install --cask codex-tracker-desktop
```

## Updating a cask

1. Update the version and SHA256 values in the relevant `Formula/` and `Casks/` files.
2. Commit and push.
